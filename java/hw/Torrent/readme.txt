Тут есть дополнительный функционал на клиентской части.
Пользовательский интерфейс показывает информацию о приложении и авторе,
также все файлы, которые можно загрузить, отображаются в таблице
(там видно название, директорию, где файл находится, его размер и папку, куда файл
был загружен, если пользователь уже скачал файл).
Кнопка удаления позволяет удалить файл из директории, куда пользователь его скачал.
Поле для поиска позволяет осуществлять фильтрацию файлов по вхождению искомой строки
в название.
Сервер работает как консольное приложение. Не надо запускать бат файл как бат файл,
он невовремя закрывает консоль, лучше эту команду ввести в терминал. 
Еще не надо запускать например его на линуксе а клиент на винде, все ломается.
Вообще... запустите лучше в идее....
